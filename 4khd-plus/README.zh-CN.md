# 4KHD Plus

[English](./README.md)

为 [4KHD](https://www.4khd.com/) 及其镜像站添加阅读器模式的用户脚本。

## 功能

- **阅读器模式** — 全屏单图浏览，支持键盘/鼠标/滚轮导航
- **智能进入** — 从当前视口可见的图片位置进入阅读器
- **跨页加载** — 自动拉取上下页图片，无缝浏览全部页面
- **缩略图滚动条** — 可拖拽滚动条，带缩略图预览面板，快速跳转
- **自动播放** — 幻灯片模式，间隔可调
- **悬浮控件** — 可隐藏的悬浮按钮与设置面板

## 支持站点

`4khd.com` · `xxtt.ink` · `uuss.uk` · `ssuu.uk`

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 将 `dist/4khd-plus.user.js` 作为用户脚本安装

## 构建

```bash
npm install
npm run build
```
